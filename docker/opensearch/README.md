# Meadowlark OpenSearch Container

This images was created to support loading [Ed-Fi
Meadowlark](https://github.com/Ed-Fi-Exchange-OSS/Meadowlark) deployments using
OpenSearch in cloud environments (e.g. Azure) where environment variables with
`.` in the name are not allowed. This image is appropriate for development and
testing, but has not been tuned for production usage.
